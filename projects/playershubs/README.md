# PlayersHubs

## Descripción

App de deportes para organizar quedadas de fútbol, gestionar equipos, y conectar jugadores con equipos a nivel profesional. Permite a equipos buscar jugadores y a jugadores encontrar equipos.

## Estado

**Activo — rehaciendo desde cero**

La versión actual está desplegada en Google Cloud y funciona, pero fue construida desde cero sin una arquitectura sólida. Se está reconstruyendo completamente.

- Base de datos: Railway
- Desarrollo: Python / VS Code
