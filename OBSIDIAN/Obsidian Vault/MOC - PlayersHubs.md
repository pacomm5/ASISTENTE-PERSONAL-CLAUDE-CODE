## Visión del producto

PlayersHubs es una plataforma para organizar partidos de fútbol, gestionar equipos y conectar jugadores. El foco está en la experiencia social y en construir una comunidad activa en torno al deporte amateur.

---

## Producto

Líneas funcionales actuales o previstas:
- Creación y gestión de partidos
- Perfiles de jugadores con historial y reputación
- Gestión de equipos
- Búsqueda de jugadores y partidos cercanos
- Ligas o competiciones informales
- Métricas de rendimiento básicas
- Sistema de valoraciones entre jugadores

---

## Comunidad

- Modelo social: jugadores, equipos, organizadores
- Incentivos para la participación activa
- Gestión de reputación y confianza dentro de la plataforma
- Posibles contenidos generados por la comunidad

## Stack técnico

Ver [[Proyectos/PlayersHubs#Parte técnica|PlayersHubs — Parte técnica]] para el detalle.

Resumen:
- Frontend: framework moderno (React o similar)
- Backend: API REST o GraphQL
- Base de datos: PostgreSQL o similar
- Infraestructura: Docker, Railway
- Desarrollo asistido con Claude Code
