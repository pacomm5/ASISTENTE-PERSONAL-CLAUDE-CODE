# Claude Design — Manual introductorio

*Fecha: 2026-04-20*

---

## La interfaz principal

En la barra superior tienes 4 pestañas:

- **Recent** — tus proyectos recientes
- **Your designs** — todos tus diseños guardados
- **Examples** — proyectos de ejemplo para inspirarte
- **Design systems** — tus sistemas de diseño configurados

---

## Panel izquierdo: Crear nuevo proyecto

**Prototype**
Para crear interfaces interactivas, apps, dashboards, mockups web. Dos modos:
- *Wireframe* — esquema estructural, sin estilos, para definir layout
- *High fidelity* — diseño visual completo, con colores, tipografía y componentes reales

**Slide deck**
Para crear presentaciones.

**From template**
Parte de plantillas prediseñadas para acelerar el proceso.

**Other**
Proyectos libres: one-pagers, landing pages, assets de marketing.

---

## Cómo crear un proyecto paso a paso

1. Escribes el nombre del proyecto en el campo *Project name*
2. Eliges el tipo (Wireframe o High fidelity)
3. Pulsas **Create**
4. Entras al editor donde describes con lenguaje natural lo que quieres y Claude lo construye

Desde dentro del editor puedes:
- Hacer comentarios inline sobre elementos concretos
- Editar texto directamente
- Usar sliders/knobs para ajustar colores, spacing, tipografía
- Pedir cambios conversando con Claude

---

## Set up design system

El paso más potente. Si configuras tu sistema de diseño (colores, tipografía, logo, componentes), todos los proyectos que crees después usarán esa identidad automáticamente. No tendrás que especificar el estilo cada vez.

Puedes subir tu codebase, archivos de diseño o definir los parámetros manualmente.

---

## Empezar

El proyecto "Learn about Claude Design" que aparece en pantalla al entrar es una guía interactiva. Recomendado como primer paso: 10-15 minutos y tienes la base completa.
